jquery-notify by jhuet

Yet another jQuery plugin to handle notifications.

Dependencies

This is a jQuery plugin so you'll need it. I created this plugin as of 1.6.1 version but it should work on older ones.

Install

Just add both jquery.notify.js and jquery.notify.css to your HTML head.
